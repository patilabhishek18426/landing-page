# Project Information: Tesla Model 3 Fan Page

# This project is a responsive static website created using HTML and CSS. It showcases information about the Tesla Model 3, highlighting its key features, design, and performance. The website includes a navigation bar, hero section, features section, and a footer, all styled with modern CSS gradients and layouts.

# The main purpose of this project is to practice web design basics, improve layout skills, and understand responsive design. It demonstrates the use of flexbox, grid, media queries, and semantic HTML to create a visually appealing and user-friendly webpage
